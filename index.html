Notas del Curso Git & Github

  • Para iniciar debemos posicionarnos con la terminal en la carpeta de nuestro proyecto, una vez que estemos en la raiz del proyecto utilizamos

    $ git init

  • Ahora git ya rastrea cambios en tu proyecto, agregar un archivo a la etapa de "staging" debemos usar el siguiente comando

    $ git add <archivo(con extension)>

  • Para agregar todos los cambios a la etapa de "staging" debemos agregar "." al final, como el siguiente comando

    $ git add .

  • Para hacer commit y guardar los cambios en el disco duro debemos utilizar el siguiente comando

    $ git commit -m "aqui va un mensaje para recordar cambios"

  • Podemos hacer commit de archivos que previamente hemos agregado con 'add' con el siguiente comando

    $ git commit -am "aqui va un mensaje para recordar cambios"

  • Para ver como era el proyecto/archivo en un commit pasado podemos usar el siguiente comando

    $ git checkout <numero de commit>

  • Una vez que estemos listos para guardar la informacion en nuestro repositorio web, hacemos commit de los ultimos cambios y despues

    $ git push origin master
